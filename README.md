# CodeAlpha Projects

This repository contains various full-stack web applications built with Node.js, Express, and Vanilla Frontend.

## 🚀 Projects Included:

### 1. [LuxeShop (E-commerce Website)](./ecart%20website/)
A premium e-commerce platform for electronics.
- **Port**: 3000
- **Features**: Product listings, shopping cart, user auth, and order processing.

### 2. [TaskFlow (Project Management Tool)](./project%20management%20tool/)
A collaborative task management system with a Kanban board.
- **Port**: 3001
- **Features**: Drag-and-drop task tracking, project dashboard, and secure authentication.

## 🛠️ How to Run
Each project has its own `server.js`. To run any of them:
1. Navigate to the project folder.
2. Run `npm install`.
3. Start the server with `node server.js`.
